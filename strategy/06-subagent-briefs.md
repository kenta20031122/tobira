# Tobira Subagent Briefs

Use these prompts after reading `00-context.md` through `05-initial-strategy-draft.md`.

## Brief 1: ICP and messaging

Task:

Recommend whether the current draft ICP is correct for the next 90 days. If not, propose a better one. Then refine the value proposition into:

- one homepage headline
- one pricing headline
- three proof points

Constraints:

- Stay within the current product shape
- Do not assume a new app, marketplace, or concierge model
- Optimize for near-term acquisition and conversion

Output:

1. Recommended ICP
2. Refined message hierarchy
3. Risks
4. How to validate in the next 30 days

## Brief 2: Activation and funnel

Task:

Evaluate the current recommendation that `planner_completed` is the main activation event and that the biggest bottleneck is the jump from discovery to authenticated planner usage.

Constraints:

- Use the current discover, email gate, signup, planner, and pricing flow
- Focus on changes that are realistic within the next 90 days

Output:

1. Agreement or disagreement with the activation event
2. Biggest drop-off point
3. Top 3 funnel fixes
4. Event instrumentation needed to prove the case

## Brief 3: Monetization

Task:

Evaluate whether Tobira Pro should be sold primarily as planning confidence rather than as access to premium hidden gems.

Constraints:

- Use current pricing, planner gating, and premium spot logic
- Recommend only monetization changes that fit the existing product

Output:

1. Primary Pro promise
2. Best free-to-paid boundary
3. Three pricing or paywall experiments
4. Key risk if the recommendation is wrong

## Brief 4: Acquisition channels

Task:

Evaluate whether SEO + Instagram should be the top channels for the next 90 days, with Pinterest as a secondary repurposing test.

Constraints:

- Use current blog, guides, and Instagram content system
- Avoid channel advice that depends on a large paid budget

Output:

1. Channel ranking
2. Best landing page for each channel
3. Top 3 content pillars
4. What to deprioritize

## Brief 5: Execution plan

Task:

Turn the strategy draft into a 6-week execution plan across product, content, and measurement.

Constraints:

- Small-team realistic
- No major replatforming
- Must define owners, deliverables, and KPIs

Output:

1. Week-by-week plan
2. Dependencies
3. Risks
4. What should be measured every week
